## Analisador de videos

Projeto em Python que recebe o link de um vídeo do YouTube e gera
automaticamente um relatório com os pontos mais importantes do conteúdo.

A aplicação extrai a transcrição do vídeo e utiliza inteligência
artificial para resumir as informações principais, facilitando o
entendimento do conteúdo sem precisar assistir ao vídeo completo.

------------------------------------------------------------------------

##  Funcionalidades

-   Receber link de vídeo do YouTube
-   Extrair transcrição automática do vídeo
-   Processar o conteúdo em Python
-   Gerar um relatório com os pontos mais importantes
-   Exibir ou salvar o relatório em arquivo

------------------------------------------------------------------------

##  Tecnologias utilizadas

-   Python
-   Requests
-   YouTube Transcript API
-   API de Inteligência Artificial
-   JSON
